Этот репозиторий содержит мои пет-проекты в сфере machine learning. Список всех проектов и 
их краткое описание представлен ниже:

1. defining_a_natural_disaster. Проект содержит нейронную сеть, которая определяет, 
   является ли введённый текст на английском языке сообщением о стихийном бедствии или нет.

   В процессе разработки проекта использовались следующие технологии:
   1.1 Нейронная сеть написана на языке python с использованием библиотеки tensorflow.
   1.2 Для отображения работы модели используется сайт, который разработан с использованием React и Flask.
   1.3 Для деплоя использовался Doker.
